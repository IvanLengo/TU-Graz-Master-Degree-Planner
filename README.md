# TU Graz Master Degree Planner

Offline university progress tracker built inside Obsidian using Meta Bind and JavaScript to create interactive study-planning and semester-management tools.

---

## Features

- ECTS tracking
- Semester planning
- Difficulty estimation
- Course organization
- Offline-first workflow
- Interactive Markdown dashboard

---

## Requirements

Before using the planner, install:

- Obsidian
- Meta Bind plugin
- JS Engine plugin

---

## Usage

1. Download or clone the repository
2. Open the folder as an Obsidian vault
3. Enable Community Plugins
4. Enable:
   - Meta Bind
   - JS Engine
5. Open `Master.md`
